1.Purpose

This document formally records the completion and handover of the Choccoladd website project from WSS Devlabs to Newbox Greens Ventures. It lists all deliverables provided, the assets and accounts transferred to the client's ownership, and serves as the formal project sign-off. Upon signature by both parties, the project is considered complete and accepted.

2.Project Overview

WSS Devlabs was engaged to design, develop, and deploy the official brand website for Choccoladd — a premium Malaysian cocoa chocolate brand by Newbox Greens Ventures, in collaboration with Lembaga Koko Malaysia. The website showcases the full product range, enables direct customer ordering via WhatsApp, and provides company, location, and contact information, with full search-engine readiness for the brand's official domain.

6.Code Repository & Integration Reference

6.1 Source Code Location

The complete source code is stored in the client's GitHub repository: github.com/choccoladd/LandingPage. Vercel is connected to this repository — any update pushed to the main branch is automatically built and deployed to www.choccoladd.com within minutes, with no manual deployment steps required.

7.Post-Handover Recommendations

  • Content updates — product names, prices, descriptions, and images are managed in the site's data files; contact WSS Devlabs or any developer for updates.
  • Domain renewal — keep choccoladd.com renewed with the registrar to avoid downtime.
  • Google Business Profile — registering the business on Google Maps is recommended to strengthen local search presence.
  • Search Console — indexing statistics and search performance can be monitored at search.google.com/search-console.
